# BitTree
二叉树学习笔记
1.定义一个BitTree类型的结构体，用于表示二叉链表中的结点。其中包含3个字段：ElemType类型变量data，用于保存结点数据；node类型（递归定义）指针变量lchild和rchild，用于指向结点的左右孩子。

2.调用CreBiTree函数，用户输入二叉树的先序序列，用#表示空链域，建立二叉树，函数返回值为根节点的指针。

3.根据用户需求，输入数字0~9执行如下操作：

（1）输入1，调用PreOrder函数，传入参数根节点指针，输出二叉树先序遍历；

（2）输入2，调用InOrder函数，传入参数根节点指针，输出二叉树中序遍历；

（3）输入3，调用PostOrder函数，传入参数根节点指针，输出二叉树后序遍历；

（4）输入4，调用LeafCount函数，传入参数根节点指针，输出叶子结点数；

（5）输入5，调用High函数，传入参数根节点指针，输出二叉树高度。思路：某一结点的处的高度是左右孩子高度的较大值+1；

（6）输入6，调用Compare函数，传入参数根节点指针与x，输出结点值大于给定值x的结点个数；

（7）输入7，调用ChildCount函数，传入参数根节点指针与x，输出值为x的结点的孩子个数；思路：找到值为x的结点，调用Count函数统计包含该结点的子树结点总数，为左右子树结点总数和+1，最终该节点的子孙总数即为Count函数返回值-1；

（8）输入8，调用ExchangeTree函数，传入参数根节点指针，交换二叉树中所有结点的左右子树，并输出新树的先序遍历；

（9）输入9，调用Exchange_xy函数，传入参数根节点指针与x、y，交换二叉树中所有值大于x小于y的结点的左右子树，并输出新树的先序遍历；

（10）输入0，结束程序。
